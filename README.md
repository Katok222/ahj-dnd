## Description

Учебный проект в стиле Trello (Drag & Drop).

## Возможности

* 3 фиксированные колонки (TODO / IN PROGRESS / DONE)
* добавление карточек
* удаление карточек
* drag and drop:

  * внутри колонки
  * между колонками
* сохранение состояния в LocalStorage

## Технологии

* JavaScript (ES6)
* Webpack
* GitHub Pages
* GitHub Actions

## Установка и запуск

```bash
yarn install
yarn start
```

## Сборка

```bash
yarn build
```
